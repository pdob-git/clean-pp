# Clean app

Creat python application with following functionalities
    - get user credentials from cli
    - login in to database
    - get data
    - display data and save as csv or excel file

Structe code using SOLID and Clean Architecture principles. Separate modules reponsible for interactions with user, getting data from database, showing results and io operations on files

Install required libaries in currenct .\.venv environment
Ensure app has high quality code easy to maintain
Use ruff and mypy for formating and type hints check
Ensure application is inline with open closed priniciple
